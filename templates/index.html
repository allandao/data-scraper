If you are like me, I wasn't sure what an RSS feed was when first working on this project. According to Macmillian Dictionary, an RSS feed "is a file that delivers information from a website, sometimes in a summary with links to the full document, to another website. RSS feeds are used mainly to supply the latest information to websites that are regularly updated."
